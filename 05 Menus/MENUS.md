# Menus

Maps of Content (MOCs) and navigation notes. A menu is a curated index for a topic or folder — a landing page you can dock in Obsidian that surfaces everything you want at hand for a focus session.

Menus are generated with `/moc <folder>` and refined manually.

## What goes here

- Folder-level MOCs: auto-generated with `/moc`, then curated
- Topic MOCs: hand-built around a domain that spans multiple folders (e.g., "Research Workflow" pulls from `01 Raw Sources/`, `03 Wiki/`, `07 Workspaces/NotebookLM`)
- Session / focus MOCs: temporary navigation notes built for a specific deep-work session (see `/focus`)
